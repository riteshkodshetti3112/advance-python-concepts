How to Run the Project

Prerequisites

Python 3.x installed on your system
VS Code or any Python IDE (optional)

Steps to Run

1. Clone the repository:

   ```bash
   git clone <your-github-repository-link>
   ```

2. Open the project folder:

   ```bash
   cd python-backend-training
   ```

3. Run the Python file:

   ```bash
   python employee_management.py
   ```

   or

   ```bash
   python3 employee_management.py
   ```

   (depending on your system)

4. The application will start and display the menu.

5. Enter your choice to:

   * Add Employee
   * Display Employees
   * Exit

## Data Storage

* Employee records are automatically saved in `employees.json`.
* Existing records are loaded automatically when the application starts.
